# Interview-Questions-Generator



## how to run:

1. Create an Environment
```bash
conda create -n interview python-3.12.0 -y

conda activate interview
```

2. Activate the environment
```
conda activate interview
```

3. Install the requirements
```
pip install -r requirements.txt

```
